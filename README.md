# 🚀 Kaung Khant Kyaw @Kevin - Personal Portfolio

A premium, high-performance personal portfolio website built with modern web technologies. This project features a stunning aesthetic inspired by **React Bits** and minimalist dark-themed designs.

## ✨ Key Features

- **🌐 Modern UI/UX**: Dark theme aesthetic with high-contrast typography and clean layouts.
- **✨ Animated Glow Background**: Dynamic, floating glow orbs and flowing border effects inspired by [ReactBits](https://reactbits.dev/).
- **💊 Floating Pill Navigation**: A sleek, glassmorphism-style navigation bar that tracks your scroll position.
- **📊 Interactive Sections**:
  - **About**: Highlighted credentials including First-Class Honours and AI Specialist role.
  - **Projects**: Visual project cards with hover-reveal details and direct links.
  - **Tech Marquee**: An infinite scrolling showcase of the core technical stack.
- **📱 Fully Responsive**: Optimized for all devices, from ultra-wide monitors to mobile phones.

## 🛠️ Built With

- **Framework**: [React.js](https://reactjs.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS3 (Custom Design System)
- **Icons**: [DevIcons](https://devicon.dev/) & Custom SVGs
- **Typography**: Google Fonts (Fredoka & Inter)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/Kcubez/my-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
